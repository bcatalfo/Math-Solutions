▶\blacktriangleright▶ Exercise A.2.   Given an equivalence relation ∼\sim∼ on a set XXX, show that the set X/∼X/\simX/∼ of equivalence classes is a partition of XXX. Conversely, given a partition of XXX, show that there is a unique equivalence relation whose set of equivalence classes is exactly the original partition.

Solution
First, we need to show that X/∼X/\simX/∼ is a partition of XXX. By the definition of a partition we need to show that XXX is the disjoint union of the sets in X/∼X/\simX/∼. By the definition of a disjoint union we need to show that A,B∈X/∼ and A≠B  ⟹  A∩B=∅A, B \in X/\sim \text{ and } A \neq B \implies A \cap B = \emptysetA,B∈X/∼ and A=B⟹A∩B=∅ (this is the "disjoint") and that {y∈[x]:x∈X}=X\{y \in [x]: x \in X\} = X{y∈[x]:x∈X}=X (this is the "union").

Let A,B∈X/∼A, B \in X / \simA,B∈X/∼ such that A≠BA \neq BA=B. Let a,b∈Xa, b \in Xa,b∈X such that A=[a]A = [a]A=[a], B=[b]B = [b]B=[b]. Suppose there exists some c∈A∩Bc \in A \cap Bc∈A∩B. Since a∼ca \sim ca∼c and b∼cb \sim cb∼c we have a∼b  ⟹  A=Ba \sim b \implies A = Ba∼b⟹A=B, a contradiction. By contradiction there exists no c∈A∩Bc \in A \cap Bc∈A∩B, so we conclude that A∩B=∅A \cap B = \emptysetA∩B=∅, the "disjoint" part of the "disjoint union".

Let y∈{y∈[x]:x∈X}y \in \{y \in [x]: x \in X\}y∈{y∈[x]:x∈X}, then for some x∈Xx \in Xx∈X we have y∈[x]  ⟹  y∼x  ⟹  y∈Xy \in [x]\implies y \sim x \implies y \in Xy∈[x]⟹y∼x⟹y∈X, so {y∈[x]:x∈X}⊆X\{y \in [x]: x \in X\} \subseteq X{y∈[x]:x∈X}⊆X. Let x∈Xx \in Xx∈X, then x∈[x]x \in [x]x∈[x], so X⊆{y∈[x]:x∈X}X \subseteq \{y \in [x]: x \in X\}X⊆{y∈[x]:x∈X}. Therefore {y∈[x]:x∈X}=X\{y \in [x]: x \in X\} = X{y∈[x]:x∈X}=X, the 'union' part of the "disjoint union".

Conversely, let C\mathscr{C}C be a partition of XXX. Let ∼\sim∼ be a relation on XXX such that a∼b  ⟺  ∃U∈C s.t. a,b∈Ua \sim b \iff \exists U \in C \text{ s.t. } a, b \in Ua∼b⟺∃U∈C s.t. a,b∈U. We need to show that ∼\sim∼ is an equivalence relation. Let a,b,c∈Xa, b, c \in Xa,b,c∈X. It is trivial that a∼b  ⟹  b∼aa \sim b \implies b \sim aa∼b⟹b∼a. If a∼ba \sim ba∼b and b∼cb \sim cb∼c, then there must exist U,V∈CU, V \in \mathscr{C}U,V∈C such that a,b∈Ua, b \in Ua,b∈U and b,c∈Vb, c \in Vb,c∈V. Since U∩V≠∅U \cap V \neq \emptysetU∩V=∅ we must have U=VU = VU=V (this is the contrapositive of the disjoint rule), so a∼ca \sim ca∼c. Therefore ∼\sim∼ is an equivalence relation as desired. □\square□
