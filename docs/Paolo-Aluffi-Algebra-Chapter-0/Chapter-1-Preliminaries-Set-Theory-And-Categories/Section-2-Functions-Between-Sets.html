2.4. ▹\triangleright▹ Prove that 'isomorphism' is an equivalence relation (on any set of sets). [§4.1][\S 4.1][§4.1]
Solution
Let SSS be a set of sets and A,B,C∈SA, B, C \in SA,B,C∈S. Define ∼\sim∼ as X∼Y  ⟺  ∃f:X→YX \sim Y \iff \exists f: X \to YX∼Y⟺∃f:X→Y, fff bijective. We now check that the properties of an equivalence relation apply.

  • reflexivity: A∼AA \sim AA∼A
    This is clear by considering idA\text{id}_AidA​, which is obviously a bijection.
  • symmetry: A∼B  ⟹  B∼AA \sim B \implies B \sim AA∼B⟹B∼A
