As the hint suggests, let's start by working out the case n=5n=5n=5. We need to show that any arrangement of the parentheses can be rearranged to

F=(((f5f4)f3)f2)f1.F = (((f_5f_4)f_3)f_2)f_1.F=(((f5​f4​)f3​)f2​)f1​.

Here is the basic strategy: first we want to isolate f5f_5f5​, then for the rest of it isolate f4f_4f4​ and shift, so on and so forth, let me give an example.

Say you start with

F′=(f5f4)(f3(f2f1)),F' = (f_5f_4)(f_3(f_2f_1)),F′=(f5​f4​)(f3​(f2​f1​)),

then by writing

F3=f3(f2f1)F_3 = f_3(f_2f_1)F3​=f3​(f2​f1​)

we can subsitute to get

F′=(f5f4)F3F' = (f_5f_4)F_3F′=(f5​f4​)F3​

and then use the associative property to get

F′=f5(f4F3).F' = f_5(f_4F_3).F′=f5​(f4​F3​).

Next we can look at F3F_3F3​ and rewrite it as

F3=f3F2,F2=f2f1F_3 = f_3F_2, \enspace F_2 = f_2f_1F3​=f3​F2​,F2​=f2​f1​

so we can plug back into F′F'F′ to get
